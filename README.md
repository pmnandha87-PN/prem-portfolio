# Prem Nandha Portfolio

A modern personal portfolio for Prem Nandha, an aspiring Full Stack Developer and AI enthusiast from Ahmedabad, Gujarat.

## Open the site

Open `index.html` in a browser. The portfolio is self-contained and uses CDN-hosted React, Tailwind CSS, Babel, and Lucide icons, so it does not require a local build step.

## Included

- Responsive React portfolio page
- Dark and light mode toggle
- Animated hero background and typing effect
- Sticky navigation and smooth scrolling
- Animated skill bars and reveal transitions
- Project, certification, interests, education, and contact sections
- Printable/downloadable resume page at `resume.html`

## Update links

The LinkedIn URL is set to `https://www.linkedin.com/in/prem-nandha`. Replace it in `index.html` and `resume.html` if the live profile uses a different slug.
